# Feature Ownership Model

Phase 1 ownership foundation:

| Feature | Ownership |
|---|---|
| Home | Repository landing dashboard, mode selection, summary, activity, and quick navigation. |
| Command Center | Repository-level assessment only. |
| Executive | Business and management metrics only. |
| Job 360 Analysis | Single job deep analysis only. |
| Converter | Migration and conversion only. |
| Documentation | Generated documents only. |
| Testing | Validation and testing only. |
| Settings | Configuration, scoring, AI guidance, templates, and governance. |

Mode visibility:

| Mode | Visible Navigation |
|---|---|
| Assessment | Home, Command Center, Executive, Job 360 Analysis, Documentation, Settings |
| Migration | Home, Converter, Documentation, Testing, Settings |
| Assessment + Migration | Home, Command Center, Converter, Executive, Job 360 Analysis, Documentation, Testing, Settings |
