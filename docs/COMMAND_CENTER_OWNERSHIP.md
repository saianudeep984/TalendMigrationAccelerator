# Command Center Ownership

Command Center is the repository-level assessment hub.

Owned sections:
- Repository Overview
- Inventory Summary
- Readiness Assessment
- Complexity Analysis
- Risk Analysis
- Dependency Analysis
- AI Insights
- Migration Assessment Summary

Ownership rules:
- Repository readiness is owned by Command Center.
- Repository complexity is owned by Command Center.
- Repository risk is owned by Command Center.
- Executive owns business and management metrics only.
- Converter owns migration and conversion only.
- Job 360 owns single-job analysis only.
- Documentation owns generated documents only.
- Testing owns validation and testing only.

Internal navigation supports:
- Overview
- Inventory
- Readiness
- Complexity
- Risks
- Dependencies
- AI Insights
- Migration Assessment
